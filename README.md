# Python Nordeste 2019

## Instalação

Para rodar o projeto é necessário instalar o [Parcel](https://parceljs.org/getting_started.html) globalmente.

```
npm install -g parcel-bundler
```

Instale as dependências do projeto.

```
npm install
```

## Desenvolvimento

```
npm run dev
```

## Build

```
npm run build
```
